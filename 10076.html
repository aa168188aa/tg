Mac 上的 Telegram 与 Telegram Lite：有什么区别？

当您在 Mac App Store 上搜索Telegram客户端时，您会发现同一应用程序的两个版本 - Telegram和Telegram Lite。这可能会令人困惑，因为他们都是官方客户。在本文中，我们将通过比较 macOS 上的 Telegram 与 Telegram Lite 应用程序、它们的功能以及 CPU 和 RAM 使用情况来帮助消除混淆。


Mac 上的 Telegram 与 Telegram Lite：比较

与 Windows 或 Linux 不同，macOS 有两个版本的 Telegram 应用程序。一种简称为Telegram，另一种称为Telegram Lite。尽管它们的工作原理相似，但它们具有不同的界面和功能集。

Telegram 与 Telegram Lite：用户界面

首先，让我们比较两个应用程序的 UI 或用户界面的差异，看看它们在主聊天页面和设置菜单的外观方面有何不同。

电报应用程序用户界面


Telegram 应用程序的 UI 与 Telegram 应用程序的 iPhone 和 iPad 版本几乎相同。如果您在 iPhone 上使用过 Telegram，那么从主页到底部快捷方式和设置菜单，Telegram 应用程序将会看起来很熟悉。它也是用 Swift（macOS 的本机编码语言）编写的。

Telegram Lite 用户界面


Telegram Lite 是 Telegram 的统一桌面版本，可在 Windows 和 Linux 上使用。这就是为什么它是用 C++ 编写的，并且缺乏上面讨论的 Telegram 应用程序的奇特功能。但该布局更适合桌面使用，并且资源略少，使其在性能和稳定性方面具有优势。

Telegram 与 Telegram Lite：特点

现在让我们看看 telegram 和 telegram lite 提供的所有功能，并比较哪一个应用程序的功能更丰富。

电报应用程序功能

  • 聊天布局与 iOS 类似。

  • 它会在一个较小的窗口中打开。

  • 底部的快速访问快捷方式。

  • 支持 macOS 手势控制。

  • 能够创建视频消息。

  • 专属系统深色模式。

  • 启用端到端加密秘密聊天的选项。

  • 渐变颜色主题。

  • 即时查看

  • 非气泡模式。

  • 更多动画和更大的表情符号。

  • 共享当前位置的选项。

Telegram Lite 功能

  • 导出聊天记录。

  • 桌面友好的界面。

  • 清除付款和运输信息的选项。

  • 硬件加速视频解码。

  • 禁用所有动画的选项。

  • 性能稳定、流畅。

由此可见，macOS Telegram 应用程序的功能更加丰富，并且将提供与 iOS 和 iPad 上可用的其他版本的 Telegram 应用程序类似的体验，而 Telegram Lite 主要针对稳定的性能。

Telegram 与 Telegram Lite：CPU、RAM 和电池使用情况

说到资源消耗，您肯定会认为名为 Telegram Lite 的应用程序比常规 Telegram 应用程序要求更低。最初，我们也有同样的想法，但心存疑虑，因此我们启动了 Activity Monitor 来检查两个应用程序的实际资源消耗情况。

CPU使用率

CPU 使用率是程序在任何给定时间使用的 CPU 容量的百分比。 CPU 使用率越低越好。相比之下，Telegram 应用比 Telegram Lite 消耗更多的 CPU 资源，如下图所示。


Telegram Lite 在前台使用时也不甘落后。但这里明显的赢家是 Telegram Lite，因为它的 CPU 使用率是两者中最少的。

内存使用情况

至于 RAM 使用情况，令人惊讶的是，两个版本的 Telegram 消耗的 RAM 量几乎相同。如图所示，两者都占用了超过 300MB 的 RAM 空间，Telegram 应用程序占用的空间稍多，达到了 400MB 的空间。

在某些情况下，Telegram Lite 在 RAM 使用方面超过了 Telegram App，因此我们可以说该测试的结果是两个应用程序之间的平局。

电池消耗

在电池消耗方面，我们观察了 2020 年配备 Touchbar 的 MacBook Pro 上这两个应用程序的平均 12 小时使用情况，结果符合预期。常规 Telegram 应用程序比 Telegram Lite 消耗更多电量。

请注意，我们在测试过程中同时运行这两个应用程序。因此平均消耗不应受到应用程序启动时间的影响。
